WeSport

This project was built using Firebase (Auth, UI, Realtime database) , uses Loaders, Notifications, Librarries such as Retrofit, Glide, Eventbus.

This app utilizes core Android user interface components and fetches movie information using themoviedb.org web API. The app provides data about popular, highest rated movies, plays videos of the movies and also stores the user's favorite movies

The mobile app uses Core components such as Fragments, Asynctask, Loaders, Sync adapters and Libraries such as Butterknife, Gson, Volley
